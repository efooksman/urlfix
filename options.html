This rule redirects Google Photos URLs to use the /u/0/ path:

{ "pattern": "photos\\.google\\.com/([^u]|$)", "replacement": "photos.google.com/u/0/$1" }

Result: photos.google.com/photo ‚Üí photos.google.com/u/0/photo

Use capture groups with $1, $2, etc. in replacements. Each rule needs a pattern and replacement.
